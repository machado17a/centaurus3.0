🎯 Objetivo Principal

O Centaurus é um sistema de verificação facial para controle de passageiros em aeroportos, utilizado pela Polícia Federal brasileira (RFH/DCRIM/INI/DPA/PF) para comparar a foto do documento de viagem com a face capturada em tempo real pela webcam.

📋 Funcionalidades Identificadas

1. Verificação Facial em Duas Etapas

  • Captura do Documento: Sistema captura a foto do documento (via screenshot ou webcam)
  • Captura da Pessoa: Captura a face da pessoa em tempo real através da webcam
  • Comparação: Usa algoritmos de IA (InsightFace com modelo AntelopeV2) para comparar as duas faces

2. Sistema de Classificação por Similaridade

O sistema classifica as verificações em 3 níveis:

Similaridade Status Ação
> 70% ✓ Verificado (verde) Liberação automática
65-70% ⚠ Atenção (amarelo) Atenção durante captura
< 65% ✗ Suspeito (vermelho) Chamar Policial Federal

3. Sistema de Cabines

  • Identificação por cabine (Ex: E1, E2, D1, D2)
  • E = Embarque, D = Desembarque
  • Rastreamento de qual terminal realizou cada verificação

4. Banco de Dados Criptografado

  • SQLite com criptografia AES-256 (SQLCipher)
  • Armazena todas as verificações, imagens (como BLOBs) e metadados
  • Localização: C:\ProgramData\Centaurus\verificacoes\verificacoes.db
  • Tabelas principais:
    • maquinas - Informações dos computadores
    • verificacoes - Registro de todas as verificações
    • imagens - Faces capturadas
    • casos_positivos - Verificações com alta similaridade
    • casos_suspeitos - Verificações suspeitas
    • auditoria_documentos - Verificações no modo documento

5. Modos de Operação

  • Modo Screenshot: Captura a tela para obter foto do documento
  • Modo Webcam: Captura direta pela câmera
  • Modo Documento: Validação específica para documentos

🔐 Aspectos de Segurança

  1. Criptografia: Banco de dados com senha hardcoded (não alterar após início)
  2. Auditoria: Todas as verificações são registradas com timestamp e informações da máquina
  3. Screenshots de casos suspeitos: Captura tela do resultado para posterior análise
  4. Rastreabilidade: Identificação da cabine e operador

📊 Uso Operacional

O sistema é utilizado por operadores da Polícia Federal em cabines de embarque/desembarque de aeroportos para:

  1. Verificar se a pessoa no documento é a mesma que está presente
  2. Detectar possíveis fraudes documentais
  3. Agilizar o processo de controle migratório
  4. Manter registro auditável de todas as verificações
